## DATA WAREHOUSES

### OLAP vs OLTP

Purpose: 
OLTP mostly common in backend databases
OLAP using for putting a lot of data in and discovering insights(analytical purposes)
data updates:
OLTP short, fast updates iniotiated by user
OLAP Data periodically regreshed with scheduled, long-running batch jobs
database design: 
OLTP: Normalized databases for efficiency
OLAP: Denormalized databases for analysis
